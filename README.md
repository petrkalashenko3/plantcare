# 🌿 Plantata — помощник по уходу за растениями

Веб-приложение, которое помогает не забывать ухаживать за комнатными
растениями: справочник с рекомендациями, личная коллекция, избранное и
напоминания о поливе и пересадке.

## Кому полезно
- **Новичкам** — понять, как ухаживать за конкретным растением.
- **Опытным** — быстро вести свою коллекцию и не пропускать уход.

## Возможности (план)
- 📖 Справочник растений с карточками ухода (полив, свет, пересадка, ядовитость, особенности)
- 🪴 Личная коллекция: свои растения, заметки, даты
- ⭐ Избранное
- 🔔 Напоминания о поливе и пересадке
- 🔎 Поиск и фильтры по справочнику

## Стек
Намеренно «ванильный», без фреймворков — чтобы код читался построчно:
- **HTML + CSS + чистый JavaScript** (без сборщиков и зависимостей)
- **LocalStorage** — хранение личной коллекции и настроек в браузере

Планируемые усложнения: серверная часть на Python + БД, распознавание
растения по фото через LLM, обмен растениями, рекомендации.

## Запуск
Приложение статическое, достаточно любого локального веб-сервера. Например:

    python -m http.server 8000

Затем открыть http://localhost:8000 в браузере.
